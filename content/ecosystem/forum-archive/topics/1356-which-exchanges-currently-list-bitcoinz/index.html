Don't forget about TradeSatoshi! They've listed BTCZ for a while now.

As for my experiences:

  • Crex24: Good liquidity, reasonable fees (0.1%), easy to use interface, quick withdrawals
  • Graviex: Decent liquidity, slightly higher fees (0.2%), interface is a bit confusing at first, withdrawals take a bit longer
  • STEX: Lower liquidity, higher fees (0.5%), good interface, withdrawal times vary
  • TradeSatoshi: Very low liquidity, high fees (0.8%), basic interface, slow withdrawals

I personally use Crex24 for most of my BTCZ trading.

Thanks for starting this thread! It's always good to have this information in one place.

Just to add to what others have said, we're currently in talks with several other exchanges for potential listings. I can't share specific names yet, but we're focusing on getting BTCZ listed on exchanges with higher volume and better user experience.

One thing to note: always make sure to use the correct deposit address when sending BTCZ to exchanges. Some exchanges have had issues with deposits when users send to the wrong address type (e.g., sending to a transparent address when they should use a shielded address).

We'll keep the community updated on any new exchange listings!
