Getting Access

  1. You need to be added as a collaborator to write posts
  2. Contact the site administrator to get access
  3. Accept the collaboration invitation from GitHub
  4. You'll then be able to create and edit posts

Uploading Images

  1. Go to Images Folder

    Visit the images folder

  2. Upload Image
    • Click "Add file" > "Upload files"
    • Drag & drop your images or click to select
    • Use descriptive names (e.g., bitcoin-update-2024.jpg)
    • Click "Commit changes"
  3. Use in Blog Post

    Add image to your post using:

    ![Image description](/images/your-image-name.jpg)

Writing a Blog Post

  1. Go to the News Folder

    Visit the news folder

  2. Create a New File
    • Click "Add file" > "Create new file"
    • Name it: YYYY-MM-DD-title.md
    • Example: 2024-01-20-bitcoin-update.md
  3. Add Your Content

    Use this template:

    ---
title: "Your Post Title"
date: 2024-01-20
---

Your content here...

## You can use markdown
- For bullet points
- **Bold text**
- *Italic text*
- [Links](https://example.com)

Image Guidelines

  • Use JPG or PNG formats
  • Optimize images for web (compress if needed)
  • Use descriptive file names
  • Add meaningful alt text for accessibility
  • Recommended sizes:
    • Featured images: 1200x630px
    • In-post images: 800px wide
    • Maximum file size: 2MB

Markdown Tips

  • **bold text** for bold text
  • *italic text* for italic text
  • # Heading for headings
  • ## Subheading for subheadings
  • [link text](url) for links
  • ![alt text](/images/image.jpg) for images
  • 1. First item for numbered lists
  • - Bullet point for bullet points

Writing Guidelines

  • Keep content relevant to BitcoinZ
  • Use clear and professional language
  • Include relevant images when helpful
  • Proofread before submitting
  • Follow the formatting template
